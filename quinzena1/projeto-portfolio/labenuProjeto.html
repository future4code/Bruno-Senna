Esse foi o meu primeiro projeto para o curso da Labenu, com ele eu consegui passar no processo seletivo. Tive que concluir em uma semana todo o processo, enquanto aprendia HTML, CSS e um pouco de Javascript. Ele é responsivo utilizando uma Media Query para ser utilizavel no IphoneX. A parte de Javascript dele é uma cofirmação que utiliza o input do Email do usuario para retornar se aquele email é valido ou não. Para fazer essa verificação utilizei um Regex.


Tecnologias utilizas:


  • HTML
  • CSS
  • Js
